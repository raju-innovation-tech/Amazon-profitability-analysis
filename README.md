# DeepThought E-Commerce FDE Fellowship Assignment

## Overview

This repository contains my submission for the DeepThought E-Commerce FDE Fellowship Assignment.

The objective of this assignment was not only to perform calculations but also to demonstrate structured problem-solving, business thinking, system thinking, and the ability to use AI responsibly while minimizing hallucination risks.

The assignment follows the DeepThought philosophy:

**Question → Hypothesis → Experiment → Insight**

# AI Usage & Hallucination Control

AI was used primarily as a brainstorming and learning tool.

To reduce hallucination risk:

* Calculations were manually verified.
* Business assumptions were reviewed logically.
* Unsupported claims were rejected.
* Real-world examples were used to validate conclusions.
* Multiple metrics were analyzed instead of relying on a single metric.

---

# Key Learnings

Through this assignment, I learned that:

* Advertising performance cannot be evaluated using a single metric.
* Customer behavior, technology, and business strategy are interconnected.
* TACOS provides a broader business perspective than ACOS alone.
* Strong organic sales improve long-term profitability.
* System thinking leads to better business decisions.
